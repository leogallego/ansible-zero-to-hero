# Módulo 5: Templates y Handlers

## Objetivos de Aprendizaje

Al finalizar este módulo serás capaz de:

- Escribir templates Jinja2 con variables, filtros, bucles y condicionales
- Usar `{{ ansible_managed | comment }}` y `backup: true` en tareas de template
- Configurar cadenas de handlers con `notify` y entender cuándo se ejecutan
- Explicar por qué los templates no deben incluir timestamps ni fechas

## La Historia Hasta Ahora

El equipo necesita generar archivos de configuración para servicios y reiniciarlos cuando las configuraciones cambian. Copiar y pegar archivos de configuración no escala.
