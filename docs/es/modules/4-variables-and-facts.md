# Módulo 4: Variables y Facts

## Objetivos de Aprendizaje

Al finalizar este módulo serás capaz de:

- Definir variables en diferentes niveles de precedencia y predecir qué valor gana
- Acceder a los facts de Ansible usando notación de corchetes (`ansible_facts['key']`)
- Usar variables registradas y `set_fact` para datos dinámicos
- Depurar variables con `ansible.builtin.debug` y `ansible-navigator`
