# Ansible Zero to Hero

A progressive course teaching Ansible from first principles to production-grade automation.

Follow **Alex**, a platform engineer at **Parasol Tech**, as they discover Ansible and scale automation across their organization.

## Modules

| # | Module | Topics |
|---|--------|--------|
| 1 | Introduction to Ansible | Environment setup, ad-hoc commands, `adt` |
| 2 | Your First Playbook | Playbook anatomy, `ansible-navigator`, idempotency |
| 3 | Managing Inventory | Structured inventory, groups, host/group vars |
| 4 | Variables and Facts | Precedence, facts, registered variables |
| 5 | Templates and Handlers | Jinja2, `ansible_managed`, handlers |
| 6 | Roles and Collections | `ansible-creator`, `ade`, Galaxy, FQCNs |
| 7 | Testing Your Automation | `ansible-lint`, Molecule, `pytest-ansible`, `tox-ansible` |
| 8 | Packaging and Deployment | Execution Environments, `ansible-builder`, `ansible-sign` |
| 9 | Scaling with AAP | Controller, Hub, EDA, workflows, RBAC |

## Getting Started

Choose your lab environment:

- **Local devcontainer** — VS Code + Docker/Podman ([setup instructions](docs/en/modules/1-introduction.md))
- **Red Hat devtools sandbox** — browser-based, no local setup needed
